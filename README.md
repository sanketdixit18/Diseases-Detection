# 🏥 Disease Prediction System using Machine Learning

## 📌 Project Overview

The Disease Prediction System is a Machine Learning-based healthcare application that predicts the likelihood of multiple diseases using patient medical data. The system analyzes health-related attributes and provides predictions for:

- Diabetes
- Heart Disease
- Breast Cancer

The project compares multiple machine learning algorithms and automatically selects the best-performing model for each disease prediction task.

---

## 🎯 Objectives

- Predict diseases using structured medical datasets.
- Compare the performance of different classification algorithms.
- Build an interactive prediction system for real-time disease prediction.
- Demonstrate the practical application of Machine Learning in healthcare.

## 📂 Datasets Used

### 1. Diabetes Dataset
Pima Indians Diabetes Dataset

**Features:**
- Pregnancies
- Glucose
- Blood Pressure
- Skin Thickness
- Insulin
- BMI
- Diabetes Pedigree Function
- Age

**Target:**
- Outcome

---

### 2. Heart Disease Dataset
UCI Heart Disease Dataset

**Features:**
- Age
- Sex
- Chest Pain Type
- Resting Blood Pressure
- Cholesterol
- Fasting Blood Sugar
- Rest ECG
- Maximum Heart Rate
- Exercise Induced Angina
- Old Peak
- Slope
- CA
- Thal

**Target:**
- Heart Disease Presence

---

### 3. Breast Cancer Dataset
Wisconsin Breast Cancer Dataset

**Features:**
- Radius
- Texture
- Perimeter
- Area
- Smoothness
- Compactness
- Concavity
- Symmetry
- Fractal Dimension
- and other medical measurements

**Target:**
- Diagnosis (Malignant / Benign)

---

## 🤖 Machine Learning Algorithms

The following classification algorithms were implemented and compared:

1. Logistic Regression
2. Support Vector Machine (SVM)
3. Random Forest Classifier
4. XGBoost Classifier

The model with the highest accuracy is automatically selected and saved.

## ▶️ Running the Project

### Train Models

```bash
python disease_prediction_system.py
```

This will:
- Load datasets
- Train models
- Compare accuracies
- Save the best models

### Start Prediction System

```bash
python prediction_menu.py
```
